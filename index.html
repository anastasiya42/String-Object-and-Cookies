String Object and Cookies

String Object Methods

The String object allows you to modify or get information about strings by using String object methods and properties. The original string is "Anastasiya Pickles" and the next section uses charAt(), slice(), and replace() to get information or modify the string.

  • The charAt() method is used to identify the letter at index 7, which is "P".
  • The slice() method is used to identify the portion of the string from indexes 11-18, which is "Pickles."
  • The replace() method is used to replace the part of the string that has the value "Pickles" with "Aleksandrova."

Cookies

Cookies are small text files saved on the end user computer that are referenced when a user comes back to the site, as long as the cookie isn't deleted/expired. Once accepted, the cookie can remember important bits of user information.
